Stock Forecast

stock prediction based on sale orders

To calculate forecasted stock Odoo default only uses open stock moves. This module allows you to also take open sale orders in the Forecasted stock calculation

Forecasted Stock Field

An extra field 'Forecasted Stock' is created and added to the product and stock views.
