Account Bank Match

automatically reconcile bankstatement lines with sale orders and invoices

Match a bank statement line with an existing sales or purchase invoice or sale order. Based on extraction of a Sale Order or Invoice reference, payment amount, bank account number, date or any other rule you can define yourself. If one correct match is found automatically link and reconcile the statement line. If more matches are found you can select the best option from a list with probabilities.
